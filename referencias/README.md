# Referencias para las slides

Puedes poner aquí materiales de referencia que quieras que se usen al mejorar las diapositivas:

- **PDFs** (por ejemplo: [El pretérito simple](https://espanoltodoenuno.com/wp-content/uploads/2023/10/El-preterito-simple.pdf)): si los guardas en esta carpeta, puedo leer su estructura y contenido para alinear tablas, ejemplos y explicaciones.
- **Imágenes** de slides o libros que te gusten (tablas, diagramas, estilo visual).
- **Archivos de texto** (.txt o .md) con listas de conjugaciones, ejemplos o notas.

Así puedo seguir mejor el estilo y la profundidad que buscas sin depender solo de enlaces externos.
