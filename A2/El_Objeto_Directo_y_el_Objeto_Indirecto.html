Pronombres Combinados

Direct Object (DO) + Indirect Object (IO) together.
"Se lo dije" / "Dámelo"

The Two Players

Before combining them, we must identify them. Remember the color code.

Receiver

Indirect Object (IO)

Function

The person who receives the action (or the result of the action).

The Magic Question

¿A quién? (To whom?)

ME • TE • LE • NOS • LES

The Item

Direct Object (DO)

Function

The thing that is being verb-ed directly.

The Magic Question

¿Qué? (What?)

LO • LA • LOS • LAS

How to identify them?

Example Sentence

"Pedro compra flores para Ana."

1

Find the Verb

"Compra" (Buys)

2

Ask: ¿QUÉ compra?

"Flores" (Feminine / Plural)

DO = LAS
3

Ask: ¿PARA QUIÉN?

"Para Ana" (She)

IO = LE

The Golden Rule: Order Matters

Spanish has a strict hierarchy. You cannot change this sequence.

"People First, Things Second"

Always place the Indirect Object (The Person) before the Direct Object (The Thing).

1

First Position

The Person (IO)

ME / TE / LE...
2

Second Position

The Thing (DO)

LO / LA / LOS...
3

Final Position

The Verb

(Conjugated)

Example Construction:

Context: "Buying flowers (las) for Ana (le)"

Se + las + compra

Remember: Le → Se

The "L-L" Rule: The Transformation

Spanish speakers dislike the sound of two "L" pronouns together. It sounds repetitive.

The Problem

LE + LO

"Impossible"

Pronunciation is difficult (Cacophony).

The Fix

Change the first "L" (The Person) to "SE".

LE / LES → SE

The Result

SE + LO

"Perfect"

Smooth pronunciation.

Incorrect (English Logic)

"Yo le lo dije."

I told it to him.

Correct (Spanish Logic)

"Yo se lo dije."

Se = To him (IO) Lo = It (DO)

Where do I put them? (Placement)

The position depends on how many verbs are in the sentence.

Scenario A

One Conjugated Verb

Strict Rule: The pronouns MUST go BEFORE the conjugated verb.

Me lo compró.

NOT: "Compró me lo" (Incorrect)

Scenario B

Two Verbs (Conjugated + Infinitive/Gerund)

Flexible Rule: You have two valid options. Choose the one you prefer!

Option 1: Detached (Before)

Place them separate, before the first verb.

Te lo voy a decir.

Option 2: Attached (After)

Attach them to the end of the infinitive. Add an accent!

Voy a decírtelo.

Quick Recap

  • 1

    Order

    Person (IO) always goes before Thing (DO).

  • 2

    Rule "L-L"

    No "Le Lo". Change the first one to SE.

  • 3

    Position

    Before conjugated verbs OR attached to infinitives.
